### 2. 使用说明

* 打开index.html

  ![](img/1.png)

* 点击upload按钮，选择obj文件进行上传，推荐上传models文件夹下的bunny.obj与Fish.obj(因为参数已经设置好)

  ![](img/2.png)

* 点击orthographic按钮，并点击rerender，切换至平行投影

  ![](img/3.png)

* 点击raster按钮，并点击rerender，切换至光栅渲染

  ![](img/4.png)

* 其他参数在下面相应输入框内可设置，设置之后需点击rerender按钮，例如: 调整分辨率变成2000x2000以消除锯齿:

  ![](img/5.png)
